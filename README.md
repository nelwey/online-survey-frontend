# Online Survey Frontend

Веб-приложение для создания и прохождения онлайн-опросов. Frontend часть приложения, построенная на React, TypeScript, Tailwind CSS, shadcn/ui и TanStack Query.

## Технологии

- **React 19** - UI библиотека
- **TypeScript** - Типизированный JavaScript
- **Vite** - Сборщик и dev-сервер
- **Tailwind CSS** - CSS фреймворк для стилизации
- **shadcn/ui** - Компоненты UI
- **TanStack Query (React Query)** - Управление серверным состоянием
- **React Hook Form** - Управление формами
- **Zod** - Валидация схем
- **React Router** - Маршрутизация
- **Recharts** - Графики и визуализация данных

## Функциональность

### Основные возможности:

1. **Главная страница** - Список всех доступных опросов
2. **Создание опросов** - Создание опросов с различными типами вопросов:
   - Текстовые вопросы
   - Вопросы с одним вариантом ответа
   - Вопросы с несколькими вариантами ответа
   - Вопросы с рейтингом (оценкой)
   - Вопросы Да/Нет

3. **Прохождение опросов** - Интуитивный интерфейс для ответа на вопросы
4. **Результаты и статистика** - Просмотр результатов опросов с визуализацией данных

## Установка

1. Установите зависимости:

```bash
npm install
```

2. Создайте файл `.env` в корне проекта:

```env
VITE_API_URL=http://localhost:3000/api
```

Замените URL на адрес вашего backend API.

3. Запустите dev-сервер:

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для production

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## API Endpoints (ожидаемые от backend)

Приложение ожидает следующие API endpoints:

- `GET /api/surveys` - Получить список всех опросов
- `GET /api/surveys/:id` - Получить опрос по ID
- `POST /api/surveys` - Создать новый опрос
- `PUT /api/surveys/:id` - Обновить опрос
- `DELETE /api/surveys/:id` - Удалить опрос
- `POST /api/surveys/responses` - Отправить ответы на опрос
- `GET /api/surveys/:id/responses` - Получить ответы на опрос
- `GET /api/surveys/:id/stats` - Получить статистику по опросу

## Использование

### Создание опроса

1. Перейдите на страницу "Создать опрос"
2. Заполните название и описание
3. Добавьте вопросы, выбрав тип каждого вопроса
4. Для вопросов с вариантами ответов добавьте варианты
5. Настройте обязательность вопросов
6. Сохраните опрос

### Прохождение опроса

1. На главной странице выберите опрос
2. Ответьте на все обязательные вопросы
3. Переходите между вопросами используя кнопки "Назад" и "Далее"
4. Отправьте ответы

### Просмотр результатов

1. На странице результатов отображается:
   - Общее количество ответов
   - Статистика по каждому вопросу
   - Графики распределения ответов
   - Детальные ответы всех респондентов

## Разработка

### Добавление новых компонентов

Используйте shadcn CLI для добавления новых UI компонентов:

```bash
npx shadcn@latest add [component-name]
```

### Стилизация

Стилизация выполняется с помощью Tailwind CSS. Темы и цвета настроены в `src/index.css` через CSS переменные.

### Типы

Все типы определены в `src/types/survey.ts`. При изменении API backend, обновите соответствующие типы.
